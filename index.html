Featured Engineering Projects

Selected projects in industrial robotics, embedded systems, robotic perception, autonomous systems, and multi robot coordination.

Event Based RGB D Perception

An event based, real time RGB D sensing system combining a monochrome event camera with a TI LightCrafter 4500 projector. Rapid structured light patterns enable independent color and depth measurements at the pixel level for both static and moving scenes, achieving color sensing equivalent to 1400 fps and pixel depth detection at 4 kHz.

Event Camera Structured Light Hardware Synchronization RGB D Sensing 3D Perception ROS Qt C++

Cobotic Weld Seam Perception and Validation

Developed a point cloud processing pipeline for weld seam detection using sequential three dimensional scans captured by a robot mounted sensor. The work included geometric feature extraction, simulated ground truth generation from CAD models, scan evaluation, and analysis of calibration and viewpoint effects on seam detection accuracy.

Cobotic Welding 3D Perception Point Cloud Processing Weld Seam Detection Ground Truth Generation Open3D Python CAD and STL Processing

Additional implementation details, datasets, and system information are not presented publicly to respect confidentiality and intellectual property obligations.

Portiloop v2

An open source portable EEG platform for closed loop stimulation. I designed the version 2.2 PCB in Altium Designer and developed version 2.3 in KiCad, supporting EEG signal acquisition, onboard processing, hardware integration, and experimental prototyping.

RoboCup Middle Size League Soccer Robotics

Engineering contributions to the MRL Middle Size League robots included schematic and PCB design for the kicker system, power management and cabling, solenoid and capacitor based kicking mechanisms, and multi robot object tracking using sensor fusion and MATLAB neural networks. My involvement progressed from team member to RoboCup Technical Committee and Executive Committee roles.

RoboCup MSL Custom PCB Power Electronics Solenoid Kicker Sensor Fusion Computer Vision Multi Robot Tracking MATLAB Neural Networks

Search and Rescue with Sparsely Connected Swarms

A decentralized search and rescue system designed for robot swarms operating with sparse and intermittent communication. Robots share a distributed belief map to coordinate exploration and establish a communication relay after detecting a target, with validation in simulation and real world drone experiments.

Autonomous Planetary Exploration Robotics

A MIST Lab multi rover platform developed for exploration and mapping in lunar and Martian cave analog environments. My contribution focused on power management, electrical integration, and wiring for the onboard rover systems, supporting field testing and deployment during the IGLUNA 2021 campaign.
